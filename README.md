# 🎨 HTML & CSS UI Animation Collection

A collection of **modern UI animations and micro-interactions** built using **pure HTML and CSS**.

This repository contains small UI components and creative UI ideas that are useful for **frontend developers, UI designers, and beginners learning web development**.

Each folder includes a **separate UI animation with clean and simple code**, so you can easily understand and use it in your own projects.

---

## 📂 Project Structure

```
├── 09_mac_loading_ui
├── 10_page_not_found
├── 11_hover_preview

├── 13_text_animation
├── 14_download_button
├── 15_delete_button
├── 16_book_page_open
├── 17_loader_animation
├── 18_hamburger_toggle
└── 19_button_gradient_hover
```

Each folder contains:

```
index.html
style.css
```

---

## 💻 How to Use

1. Clone the repository

```bash
git clone https://github.com/buildwithdeep2199/yt-source-code.git
```

2. Open any folder.

3. Run **index.html** in your browser.

You can copy the **HTML and CSS code** directly into your own projects.
